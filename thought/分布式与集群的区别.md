## 分布式与集群

|      | 分布式                                                   | 集群                                   |
| ---- | -------------------------------------------------------- | -------------------------------------- |
| 概念 |                                                          |                                        |
| 简化 | 一个业务被拆成多个子业务部署在不同的服务器上，就是分布式 | 同一个业务部署在多台服务器上，就是集群 |
