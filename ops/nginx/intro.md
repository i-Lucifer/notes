### nginx和apache对比

| 特性           | nginx                      | apache                            | 说明                              |
| -------------- | -------------------------- | --------------------------------- | --------------------------------- |
| 配置           | 简单                       | 复杂                              |                                   |
| 静态请求       | 对html支持更好，内存小     |                                   | nginx3倍与apache                  |
| 动态请求       |                            | php net jsp需要进行模块解析时高效 | 比如 模块式PHP                    |
| 静态与动态场景 | 适合静态和反向代理         | 适合动态请求                      |                                   |
| 模块式PHP      | 不支持                     | 支持                              | 模块解析比转发到php-fpm少一次通讯 |
| 网络模型       | 异步，多个连接对应一个进程 | 同步多进程，一个连接一个进程      | 异步模型支持更高的连接            |
| 反向代理       | 支持                       | 未知                              |                                   |
| 负载均衡       | 支持                       | 未知                              |                                   |
